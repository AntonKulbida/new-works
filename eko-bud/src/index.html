  • Большой земснаряд – это плавательное средство, на котором размещается мощный насос для подъема смеси воды с песком (или грунтом) со дна водоема. Перед всасыванием грунт предварительно размывается мощным потоком воды. Затем такая смесь, которая называется пульта, по специальным трубопроводам или с помощью другой вспомогательной техники транспортируется в определенное место.


    Большой земснаряд применяется для:

    - добычи песка для использования в промышленных целях;

    - намыва песка для создания пляжей или грунтовой смеси для - создания набережной, островов;

    - создания береговой линии с поднятием уровня берега на водой;
